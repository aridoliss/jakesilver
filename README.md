# Silver Studios Website

A simple, modern, responsive website built using only:

- HTML5
- CSS3
- JavaScript

Designed to be hosted using GitHub Pages.

---

# Features

## Pages

### Home

Contains:

- Hero section
- Company introduction
- Highlights
- Featured projects

---

### Showcase

Portfolio section for:

- Games
- 3D Models
- Programming Projects
- Tools
- Artwork

Projects are displayed using reusable cards.

To add a new project, copy an existing project card and change:

- Image
- Title
- Description
- Link

---

### Contact

Contains:

- Company information
- Social links
- Contact form

The contact form is currently visual only.

GitHub Pages cannot process form submissions without an external service.

Possible integrations:

- Formspree
- EmailJS
- Web3Forms
- Custom backend API
